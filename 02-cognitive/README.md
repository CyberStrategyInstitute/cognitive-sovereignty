# Domain 02: Cognitive

## Definition
Attention control, deep thinking, memory, learning agility, reasoning capacity.

## Primary Threats
Attention capture (T-CT-001), cognitive load flooding (T-CT-014), information ecosystem poisoning (T-CT-007), cognitive offloading atrophy (ST-003).
